# 📸 Photography Portfolio – Prabhakar Shukla

> A modern, aesthetic, and responsive photography portfolio website to showcase my best captures.

---

## 🌟 About The Project

This is my personal **Photography Portfolio Website** where I showcase some of my best shots that feel aesthetic, memorable, and meaningful.

The goal of this project was to:
- Create a visually appealing gallery  
- Maintain a minimal yet premium UI  
- Ensure smooth performance across devices  
- Protect images using watermark overlay  

---

## 🚀 Features

✨ Clean & modern UI  
📱 Fully responsive (Mobile + Desktop optimized)  
🖼️ Interactive image gallery  
🔍 Click-to-view full image preview  
💧 Watermark overlay for image protection  
⚡ Smooth scrolling experience  
🎨 Carefully selected typography & layout  

---

## 🛠️ Tech Stack

- Frontend: Next.js  
- Styling: Tailwind CSS  
- Deployment: Vercel  
- Version Control: GitHub  

---

## 📂 Project Structure

/public  
   └── images → All photography assets  

/src  
   ├── components → Reusable UI components  
   └── app → Main pages (Next.js routing)  

/styles → Global styles

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/prabhakarshukla/photography-portfolio

# Navigate to project folder
cd PHOTOGRAPHY_PORTFOLIO

# Install dependencies
npm install

# Run development server
npm run dev
